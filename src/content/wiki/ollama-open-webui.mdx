Dieses Setup kombiniert **Ollama** (lokaler LLM-Runner) mit **Open WebUI** (ChatGPT-ähnliche Oberfläche). Ergebnis: Ein lokaler AI-Chatbot — komplett kostenlos und offline-fähig.

## Installation

### 1. Ollama installieren

```bash
# macOS / Linux
curl -fsSL https://ollama.ai/install.sh | sh

# Windows: Installer von ollama.ai
```

### 2. Modell laden

```bash
ollama pull llama3.1:8b
```

### 3. Open WebUI starten

```bash
docker run -d -p 3000:8080 \
  --add-host=host.docker.internal:host-gateway \
  -v open-webui:/app/backend/data \
  --name open-webui \
  ghcr.io/open-webui/open-webui:main
```

### 4. Browser öffnen

Gehe zu `http://localhost:3000` — fertig.

## Empfohlene Modelle

| Modell | RAM | Befehl |
|---|---|---|
| Llama 3.1 8B | 8 GB | `ollama pull llama3.1:8b` |
| Gemma 2 9B | 8 GB | `ollama pull gemma2:9b` |
| Mistral 7B | 8 GB | `ollama pull mistral` |

## Troubleshooting

- **Open WebUI findet Ollama nicht**: Prüfe ob `ollama serve` läuft
- **Zu langsam**: GPU-Beschleunigung aktivieren
- **Port belegt**: `-p 3001:8080` verwenden

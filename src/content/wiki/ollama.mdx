**Ollama** ermöglicht es, Large Language Models lokal auf dem eigenen Computer auszuführen — ohne Cloud, ohne Kosten, mit voller Datenkontrolle. Ein einfacher Befehl wie `ollama run llama3` lädt und startet ein Modell.

## Kernfunktionen

- **One-Command-Install**: `ollama run <modell>`
- **Modelllibrary**: Hunderte Modelle (Llama, Gemma, Mistral, Phi)
- **GPU-Beschleunigung**: NVIDIA CUDA, Apple Metal
- **REST-API**: OpenAI-kompatibel auf `localhost:11434`
- **Modelfile**: Eigene Konfigurationen

## Hardware

| Modellgröße | Min. RAM | Empfohlen |
|---|---|---|
| 7B Parameter | 8 GB | 16 GB |
| 13B Parameter | 16 GB | 32 GB |
| 70B Parameter | 48 GB | 64 GB |

## Verfügbare Modelle

| Modell | Parameter | Beschreibung |
|---|---|---|
| Llama 3.1 | 8B–405B | Metas Open-Source-LLM |
| Gemma 2/3 | 2B–27B | Googles effizientes Modell |
| Mistral | 7B | Europäisches Modell |
| Phi-3 | 3.8B–14B | Microsofts Kompaktmodell |

**Claude Code** ist ein KI-gestützter Coding-Assistent von Anthropic. Er arbeitet als vollwertiger Entwicklungspartner, der das gesamte Repository liest und versteht. Verfügbar als CLI-Tool, Desktop-App, Web-App und als Extension für VS Code und JetBrains IDEs.

## Geschichte

Claude Code wurde 2025 als Teil des Claude-Ökosystems eingeführt. Ursprünglich als reines CLI-Tool konzipiert, wurde es sukzessive um IDE-Integrationen und eine Desktop-App erweitert. Mit Claude 4 Opus (2026) erhielt es deutlich verbesserte Reasoning-Fähigkeiten.

## Funktionsweise

Im Kern nutzt Claude Code die Claude-Sprachmodelle (Sonnet, Opus) um Quellcode zu analysieren, zu generieren und zu refactoren. Anders als einfache Code-Completion-Tools liest Claude Code das gesamte Projektverzeichnis und baut ein Verständnis für die Architektur auf.

### Kernfunktionen

- **Kontextverständnis**: Liest das gesamte Repository ohne manuelle Kontextgabe
- **Multi-File-Editing**: Änderungen über beliebig viele Dateien gleichzeitig
- **Terminal-Integration**: Führt Befehle direkt aus (Tests, Builds, Git)
- **Git-Integration**: Commits, Pull Requests, Code Reviews
- **Agent-Modus**: Führt komplexe Aufgaben mehrstufig autonom aus

## Einschränkungen

- Erfordert Internetverbindung
- Usage-Limits im Pro-Plan bei intensiver Nutzung
- Kein kostenloser Tier für Claude Code selbst
- Gelegentlich zu ausführlich bei einfachen Aufgaben
